## 轻松上手

daruk 提供了 cli 命令可以自己定义项目模板，默认提供了 2 种，详情参考[脚手架](./tutorial/Scaffolding.md).

```bash

// 全局安装 daruk cli
$ yarn global add daruk-cli # cnpm install -g daruk-cli

// 使用daruk 提供的模板
$ daruk init daruk-example

$ cd daruk-example

$ npm run dev
```
